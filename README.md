# SDC255 Week 3 Performance Assessment

## Overview
This repository contains the Python program for the Week 3 Performance Assessment in SDC255. The program demonstrates the use of functions, parameters, return values, and program flow in Python.

The program includes three functions that are called from the `main()` function.

## Program Functions

### functionOne()
Displays the student ID in the console.

### functionTwo()
Prompts the user to enter two numbers, adds them together, and displays the result. The function returns the sum to the `main()` function.

### functionThree(sumValue)
Receives the sum from `functionTwo()`.

- If the sum is greater than 5, the program prints **"The sum is greater than 5."**
- If the sum is 5 or less, the program prints **"The sum is 5 or less."**

The function returns the numeric portion of the student ID.

## How to Run the Program

1. Open the file `performanceAssessmentFunctions.py`
2. Run the program using Python
3. Enter two numbers when prompted
4. The program will display the sum and additional messages based on the result
